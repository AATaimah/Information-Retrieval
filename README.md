## Repository Structure
- `src/` - preprocessing, indexing, retrieval, and neural reranking code
- `outputs/index/` - prebuilt index artifacts
- `trec_eval/` - evaluation tool
- `Results*` - run output files for the assignments

## Dataset Note
Large dataset files are not meant to be committed to the repository. See `A1.md` and `A2.md` for the exact local file layout expected by each assignment script.

## Notes
- `A1.md` documents the TF-IDF retrieval system.
- `A2.md` documents the neural reranking system built on top of the Assignment 1 pipeline.
- `Assignment2_Report.md` is the full write-up in submission-report format.
